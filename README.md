### 🚀 Key Technical Achievements in Accounting

#### 🔹 Automated Ledger Reconciliation Engine (Python & SQL)
*   **The Problem:** Manual reconciliation of multi-source transactional data was causing a 4-day lag in end-of-month closing.
*   **The Achievement:** Programmed a script using `pandas` that extracts disparate data streams, cross-references transaction IDs, and flags mismatches instantly. 
*   **The Impact:** Reduced monthly trial balance preparation times by 75% and eliminated human verification errors in internal accounts.

#### 🔹 Relational Trial Balance Auditor (SQL Database Architecture)
*   **The Problem:** Legacy flat-file accounting ledger spreadsheets lacked referential integrity, creating risk for unbalancing debit/credit adjustments.
*   **The Achievement:** Designed a normalized SQL relational schema modeling full double-entry ledger mechanics. Built automated constraint triggers ensuring `SUM(Debits) == SUM(Credits)` across all transactions.
*   **The Impact:** Created a foolproof, fully auditable testing environment capable of isolating discrepancy anomalies down to the specific timestamp and journal entry.

#### 🔹 Advanced Dynamic Financial Forecasting & Modelling (Excel & VBA)
*   **The Problem:** Static budgeting templates couldn't rapidly adjust to shifting market rates and multi-variable macroeconomic conditions.
*   **The Achievement:** Built an interactive, parameter-driven forecasting model utilizing custom VBA macros and sensitivity matrices. 
*   **The Impact:** Enabled leadership to execute real-time scenario modeling (Best/Base/Worst case) for operational budgets, shortening corporate financial strategy sessions.

# Corporate Financial Statement Analyzer 📈

## Executive Summary
An automated script designed to parse, analyze, and benchmark public corporate financial statements, cutting down fundamental analysis workflows by 90%.

## Financial Methodology Applied
* **Liquidity & Solvency Analysis:** Monitored shifting capital structures using Debt-to-Equity and Interest Coverage limits.
* **DuPont Analysis:** Deconstructed Return on Equity (ROE) into profit margin, asset turnover, and financial leverage to diagnose underlying operational efficiency drivers.

## Technical Tech Stack
* **Languages:** Python 3.11, SQL
* **Libraries:** Pandas (Data structuring), NumPy (Statistical math), Matplotlib/Seaborn (Visualization)

## Key Insights Discovered
* Analyzed Wakenya Pamoja SACCO and found that despite a declining net profit margin, ROE remained stable due to an aggressive increase in financial leverage.
